Goal: Create a layout where boxes are:

  • 50% width (side by side) on desktop/large screens (xl and above)
  • 100% width (stacked) on mobile/smaller screens

Solution: Use col-xl-6 for 50% on extra-large screens, defaults to 100% below xl breakpoint

Try it: Resize your browser window or use Chrome DevTools device toolbar to see the responsive behavior!
