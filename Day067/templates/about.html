About Me

This is what I do.

Hello! I'm Angela, a passionate blogger who loves to share thoughts and experiences about technology, life, and everything in between.

This blog is built with Flask, a powerful Python web framework, and demonstrates full CRUD (Create, Read, Update, Delete) functionality. Every post you see here is stored in a SQLite database and can be created, edited, or deleted through the web interface.

The rich text editing is powered by CKEditor, allowing for beautiful formatting and styling of blog posts. Bootstrap provides the sleek, responsive design you see across the site.

Feel free to explore the blog, read the posts, and see how modern web development combines backend logic with elegant frontend design!
