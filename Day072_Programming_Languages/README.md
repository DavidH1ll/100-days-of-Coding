# Day 72 - Data Visualisation with Matplotlib

## Overview
Created multi-panel visualizations of programming language popularity trends using matplotlib. Line charts, bar charts, pie charts, and subplots in a single figure.

## Key Concepts
- plt.plot(), plt.bar(), plt.pie(), plt.subplots()
- Titles, labels, legends, color maps
- Multi-panel dashboard layout

## Reflection
Matplotlib's subplot API is powerful but verbose. Building a 4-panel dashboard required careful management of axes objects. The horizontal bar chart with `invert_yaxis()` is a nice touch for ranking displays.
