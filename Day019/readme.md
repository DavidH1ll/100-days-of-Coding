# Day 19 – Turtle Events & Mini Games

## Overview
Focus: handling keyboard events and creating interactive turtle-based mini games.

## Files
- main.py – simple event handling demo (movement / key bindings).
- etchasctch.py – Etch-A-Sketch clone (arrow/WASD to draw, clear/reset).
- turtlerace.py – Turtle race (random forward movement, user bet input).

## Concepts
- `Screen().listen()` and `onkey()` for key events.
- State updates vs immediate redraw.
- Random distances for race simulation.
- Simple user input prompts (`input()`).

## Run
````powershell
python etchasctch.py
python turtlerace.py
