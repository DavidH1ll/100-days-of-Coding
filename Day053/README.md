# Day 053 - Web Scraping with Selenium

## Project Overview
Implemented web scraping techniques using both Beautiful Soup and Selenium to extract data from websites and automate form submissions. This project demonstrates data collection and browser automation.

## What I Learned
- Web scraping with Beautiful Soup
- Browser automation with Selenium
- HTML parsing and data extraction
- Working with dynamic web content
- Automated form filling

## Key Technologies
- **Beautiful Soup**: For parsing HTML content
- **Selenium WebDriver**: For browser automation
- **Requests**: For HTTP requests

## Features
- Scrape property listings from websites
- Extract structured data (prices, addresses, links)
- Automate form submissions with collected data
- Handle dynamic web content

## How to Run
```bash
pip install beautifulsoup4 selenium requests
python main.py
```

## Key Concepts
- HTML parsing and navigation
- CSS selectors and XPath
- WebDriver API
- Wait conditions and timing
- Data extraction patterns
