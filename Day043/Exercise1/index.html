Three Methods of Adding CSS

1. Inline CSS

CSS is written directly in the HTML element's opening tag using the style attribute. Best for styling a single element.

2. Internal CSS

CSS is written inside a <style> tag in the HTML document's <head> section. Best for styling a single webpage.

3. External CSS

CSS is written in a separate .css file and linked to the HTML using a <link> tag. Best for multi-page websites.
