What is External CSS?

External CSS is a method of applying styles using a separate .css file that is linked to the HTML document.

Syntax

In HTML:

<link rel="stylesheet" href="style.css">

In style.css:

h1 {
  color: green;
}

How It Works

  • CSS rules are written in a separate file with a .css extension
  • The CSS file is linked to HTML using a <link> tag in the <head>
  • The rel attribute specifies the relationship (stylesheet)
  • The href attribute points to the CSS file location
  • Styles apply to all pages that link to the CSS file

Advantages

  • Cleaner, more readable HTML
  • Styles reused across multiple pages
  • Easy to maintain and update (one file to modify)
  • Smaller HTML file sizes
  • Browser can cache CSS file for better performance
  • Separation of HTML structure and CSS styling

Disadvantages

  • Requires an additional HTTP request to load CSS file
  • Slightly more complex setup than inline CSS

Best Practices

  • Use external CSS for all production websites
  • Organize CSS files logically (main.css, utilities.css, etc.)
  • Use meaningful CSS selectors and classes
  • Keep CSS modular and reusable
  • Consider using CSS preprocessors (SCSS, LESS) for large projects
