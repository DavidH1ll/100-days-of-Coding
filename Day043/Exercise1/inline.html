What is Inline CSS?

Inline CSS is a method of applying styles directly to an HTML element using the style attribute.

Syntax

<h1 style="color: blue;">Style me in blue!</h1>

How It Works

  • The style attribute goes inside the opening HTML tag
  • CSS properties and values are written as key-value pairs
  • Multiple properties are separated by semicolons
  • Example: style="color: blue; font-size: 24px;"

  • ‚úÖ Styling a single, unique element
  • ‚úÖ Quick testing or prototyping
  • ‚úÖ Temporary overrides

  • ‚ùå Difficult to maintain across multiple elements

Advantages

  • Simple for one-off styles
  • No need for separate CSS files
  • High specificity (overrides other CSS)

Disadvantages

  • Hard to maintain and update
  • Clutters HTML markup
  • Cannot reuse styles across multiple elements
  • Violates separation of concerns principle
