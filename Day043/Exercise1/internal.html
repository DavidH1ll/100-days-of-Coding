What is Internal CSS?

Internal CSS is a method of applying styles to an HTML document using a <style> tag placed in the <head> section.

Syntax

<style>
h1 {
  color: red;
}
</style>

How It Works

  • The <style> tag is placed in the HTML <head> section
  • CSS rules use selectors to target HTML elements
  • Styles apply to all matching elements in that page
  • Each rule has a selector, followed by curly braces containing properties

Advantages

  • Cleaner HTML markup (no style attributes)
  • Can style multiple elements at once
  • Easier to maintain than inline CSS
  • Good for single-page applications

Disadvantages

  • Styles are limited to one HTML document
  • Cannot be reused across multiple pages
  • HTML file size increases with CSS
  • Difficult to maintain in large projects
