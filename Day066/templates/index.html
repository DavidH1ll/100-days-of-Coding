Welcome to the Cafe API!

This REST API provides access to a database of remote-work friendly cafes, including information about WiFi availability, power outlets, seating, and coffee prices.

Base URL: http://127.0.0.1:5000

GET /search?loc={location}
Search for cafes by location. Returns all cafes in the specified area.

Query Parameters:

  • loc - Location to search for (e.g., "Peckham", "London")
Example Request:
GET /search?loc=Peckham

Example Response (Success):
{
  "cafes": [
    {
      "id": 1,
      "name": "Science Cafe",
      "location": "Peckham",
      ...
    }
  ]
}

Example Response (Not Found):
{
  "error": {
    "Not Found": "Sorry, we don't have a cafe at that location."
  }
}
POST /add
Add a new cafe to the database. Submit cafe details via form data.

Required Form Data:

  • name - Cafe name (string)
  • map_url - Google Maps URL (string)
  • img_url - Image URL (string)
  • location - Location/area (string)
  • seats - Number of seats (e.g., "20-30", "50+")
  • has_toilet - Toilet availability (boolean)
  • has_wifi - WiFi availability (boolean)
  • has_sockets - Power socket availability (boolean)
  • can_take_calls - Phone call friendly (boolean)
  • coffee_price - Coffee price (e.g., "¬£2.50")
Example Response (Success):
{
  "response": {
    "success": "Successfully added the new cafe."
  }
}
