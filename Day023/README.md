# Day 23 – Turtle Crossing Game

## Overview
Implemented a Frogger-style crossing game: player moves upward avoiding horizontally moving cars. Difficulty increases as cars accelerate.

## Files
- main.py – game loop, screen setup, key binding, level progression.
- player.py – Player (turtle) movement and reset.
- car_manager.py – Spawns cars, advances them, speed scaling.
- scoreboard.py – Level tracking and game over display.

## Run
```bash
python main.py
