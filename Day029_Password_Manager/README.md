# Day 029 - Password Manager (MyPass)

## Project Overview
Built a comprehensive password manager application using Python's Tkinter library. This project focuses on GUI development, password generation, and local data persistence.

## What I Learned
- Creating graphical user interfaces with Tkinter
- Password generation algorithms for security
- Local file-based data storage
- Input validation and error handling
- User experience design principles

## Project Structure
```
Day029/
├── MyPass/
│   ├── src/
│   ├── requirements.txt
│   └── README.md
└── README.md
```

## Features
- **Password Generation**: Generate strong, complex passwords
- **Local Storage**: Save passwords securely in local files
- **Field Validations**: Ensure proper input formatting
- **User-Friendly Interface**: Clean Tkinter GUI

## How to Run
```bash
cd MyPass
pip install -r requirements.txt
python src/main.py
```

## Key Concepts
- GUI programming with Tkinter
- Password security best practices
- File I/O operations
- Data validation
