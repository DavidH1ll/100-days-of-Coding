# Image Placeholders for A Hotel Website

This folder contains placeholder images for the hotel website features.

## Image Requirements

### bedroom.jpg
- **Size:** 800x800px (1:1 aspect ratio)
- **Content:** Luxurious hotel bedroom with modern decor
- **Suggested Source:** Unsplash - https://unsplash.com/s/photos/luxury-hotel-bedroom

### food.jpg
- **Size:** 800x800px (1:1 aspect ratio)
- **Content:** Gourmet breakfast or elegant dining spread
- **Suggested Source:** Unsplash - https://unsplash.com/s/photos/hotel-breakfast

### pool.jpg
- **Size:** 800x800px (1:1 aspect ratio)
- **Content:** Infinity pool with tropical or ocean view
- **Suggested Source:** Unsplash - https://unsplash.com/s/photos/infinity-pool

## Quick Setup

Replace these placeholders with actual images:

1. Download free high-quality images from Unsplash
2. Resize to 800x800px for optimal loading
3. Name them exactly: bedroom.jpg, food.jpg, pool.jpg
4. Place in this images/ directory

## Alternative: Using Unsplash Directly

The CSS already uses Unsplash URLs for background images.
You can also use Unsplash URLs directly in the HTML:

```html
<img src="https://images.unsplash.com/photo-1566665797739-1674de7a421a?w=800&h=800&fit=crop" alt="Bedroom">
<img src="https://images.unsplash.com/photo-1567137098126-5a6be6d48c44?w=800&h=800&fit=crop" alt="Food">
<img src="https://images.unsplash.com/photo-1576013551627-0cc20b96c2a7?w=800&h=800&fit=crop" alt="Pool">
```
