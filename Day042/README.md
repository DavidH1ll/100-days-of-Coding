# Day 42 - Birthday Invite Website

## Project Overview
Create a fun and functional birthday invitation website that combines essential HTML elements. This retro 90s-style project brings together images, lists, links, and headings to build an interactive party invitation page.

## Learning Objectives
- Combine multiple HTML elements into a cohesive project
- Master the `<img>` tag with source and alternative text for accessibility
- Create unordered lists (`<ul>`) with list items (`<li>`)
- Implement anchor tags (`<a>`) with href attributes
- Build a practical, user-friendly webpage
- Practice semantic HTML structure

## How to Use
1. Open `index.html` in any web browser
2. View the birthday invitation website
3. Click the Google Maps link to see the party location
4. Customize with your own party details:
   - Change the date and time
   - Update the venue description
   - Modify the guest list items
   - Replace the image URL with your own
   - Update the Google Maps link to your location

## Key Takeaways
- HTML elements work together to create meaningful content
- Images require both `src` (source) and `alt` (alternative text) attributes
- Unordered lists organize information in a scannable format
- Anchor tags connect to external resources like Google Maps
- Semantic HTML improves readability and accessibility
- A well-structured invitation is both functional and welcoming

## Accessibility Notes
- Alt text on images describes content for screen readers
- Semantic heading hierarchy (h1, h2) helps users navigate
- List structure makes information easily scannable
- Link text clearly indicates destination (Google Maps)
