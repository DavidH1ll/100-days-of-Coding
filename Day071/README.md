# Day 71 - Data Exploration with Pandas

## Overview
First day of the Data Science module. Explored a synthetic dataset of college majors and their salary/employment outcomes using pandas for data loading, inspection, cleaning, grouping, and filtering.

## Key Concepts
- DataFrame, Series
- .head(), .tail(), .shape, .columns, .isna(), .describe()
- .groupby(), .sort_values(), .value_counts()
- Boolean filtering, missing value handling

## Reflection
Pandas makes data exploration feel like querying a database with Python. The `.describe()` + `.groupby()` combo answered 90% of my analytical questions in two lines. The deliberate NaN was a good reminder that real data is never clean on arrival.
