# Day 73 - Aggregate & Merge Data with Pandas

## Overview
Simulated Lego dataset analysis demonstrating merge, groupby aggregation, pivot tables, and multi-index operations. Two related DataFrames (sets and themes) are joined and analyzed.

## Key Concepts
- .merge(), .groupby().agg(), .pivot_table()
- Multi-index operations
- Missing value handling

## Reflection
The merge + groupby + agg pipeline is the pandas power combo. Understanding the difference between inner/outer/left joins is critical when working with related tables. The pivot table flattened year-by-theme data elegantly.
